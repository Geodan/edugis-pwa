Http en https

Gebruik voor de links altijd https, niet http. Als het echt niet anders kan (omdat er geen https variant is), dan moet ook http worden gebruikt voor de link naar deze sbs.html pagina, dus http://kaart.edugis.nl/sbs.html. Dit werkt niet altijd goed, omdat sommige browsers onthouden dat sbs.html ook via https te bereiken is en dan wordt daar automatisch omgeschakeld. Vanaf een pagina die met https werkt, worden http links niet toegestaan.
